# ioihw20-helper

### Usage

1. 下载浏览器插件 Tamper Monkey，新建脚本。
2. 复制 [`./userscript.js`](https://raw.githubusercontent.com/memset0/ioihw20-helper/master/userscript.js) 的全部内容到新建脚本内，并保存。
3. **点亮右上角的 Star**

### Feature

* 替换用户名成真实姓名。
* 查找卷王。
* 好像没了。
